============================================================
MATH
============================================================
periodic_rate(annual_pct, n):
    Two-step Canadian semi-annual compounding:
    eff = (1 + r/200)**2
    return eff**(1/n) - 1

calc_pmt: standard annuity formula; accelerated variants divide monthly pmt by (n/12).
Build CalYear column in amortization df using _year_of(date).

BUGS FIXED (vs v7)
============================================================
FIX #1: Second scenario save TypeError — ternary expression
  sc_col2.success(...) if ok else sc_col2.error(...)  ← CAUSED BUG
  Replace with:
    if ok:
        sc_col2.success(...)
    else:
        sc_col2.error(...)
  Root cause: Streamlit tries to iterate the return value of a ternary
  result when it appears at statement level.

============================================================
ENTRY POINT
============================================================
app.py:
  1. st.set_page_config (must be first)
  2. Inject global CSS (with dark-mode fixes)
  3. Import mortgage_db; initialise session_state
  4. DB Connection Gate (calls get_db_connection which auto-creates tables)
  5. Load saved setup if present
  6. Render header
  7. Create tabs list
  8. Call each tab module: tab_setup.render(tabs), tab_scenarios.render(tabs), etc.

Run with:
  streamlit run app.py

Dependencies:
  pip install streamlit pyodbc pandas numpy plotly python-dateutil python-docx
